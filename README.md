Crypto wallet ui made in wpf with c# and xmal pair with working crypto price api developers.coindesk.com
and login form that can accept private key and seed phrase from another wallet or login with google,coinbase,phantom wallet
also the background has 2 different themes. (This project is still in development there will be more updates on this repository in the future.) :D
